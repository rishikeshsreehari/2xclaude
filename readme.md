# claude2x

Tells you instantly if Claude's 2x usage promotion is active right now.

🔗 **Live:** [2x.rishikeshs.com](https://2x.rishikeshs.com)  
📋 **Official details:** [Claude March 2026 Usage Promotion](https://support.claude.com/en/articles/14063676-claude-march-2026-usage-promotion)

---

**How it works:** Anthropic is doubling usage limits (March 13–27, 2026) outside peak hours — weekdays 8am–2pm ET are normal, everything else (including all weekends) is 2x. This page checks your current time against that window and shows a countdown to the next change.

Deployed on Cloudflare Pages. Single `index.html`, no dependencies.
